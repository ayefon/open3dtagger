Open 3D Tagger

Read and write NFC tags on 3D printer filament spools using the OpenTag3D open standard.

About

Open 3D Tagger turns your iPhone into a filament spool manager. Tap your phone to an NFC tag on any spool to instantly see material type, color, manufacturer, print temperatures, and more.

Write your own tags so printers and slicers can automatically configure for each spool. Edit and rewrite tags as your filament data changes.

Features

  • Read OpenTag3D NFC tags with a single tap
  • Write filament data including material, colors, temperatures, and physical properties
  • Edit and rewrite existing tags with updated information
  • Extended data support: serial numbers, drying settings, volumetric speeds, and more
  • Compatible with NTAG213, NTAG215, NTAG216, and SLIX2 tags
  • Built on the OpenTag3D v1.000 open specification
  • Fully offline — all data stays on your device

Requirements

iPhone 7 or later with NFC capability. Compatible NFC tags (NTAG213, NTAG215, NTAG216, or SLIX2) are required for reading and writing.
