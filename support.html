Support

Requirements

Open 3D Tagger requires an iPhone 7 or later with NFC capability. You also need compatible NFC tags: NTAG213 (144 bytes, core data only), NTAG215 (504 bytes), NTAG216 (888 bytes), or SLIX2 (320 bytes) for full core and extended data.

NFC scan sheet doesn't appear

Make sure you're tapping the "Scan Tag" or "Write to Tag" button, not the tab bar icon. The NFC scan sheet only appears after pressing one of these action buttons.

If you've never used NFC on this device, iOS will prompt you for permission the first time. Check Settings > Privacy & Security > NFC to ensure it's enabled.

Tag not reading

The iPhone's NFC antenna is located at the top rear of the device, near the camera module. Hold the top of your iPhone directly against the NFC tag on the spool. Keep it still for a moment while the scan completes.

If you see "No OpenTag3D data on this tag," the tag either hasn't been written with OpenTag3D data or uses a different format.

Tag too small error when writing

NTAG213 tags (144 bytes) only support core data fields. If you've enabled "Include Extended Data" in the Write form, you'll need a larger tag: SLIX2 (320 bytes), NTAG215 (504 bytes), or NTAG216 (888 bytes).

Disable the "Include Extended Data" toggle to write core-only data to smaller tags.

Temperature values seem wrong

The OpenTag3D specification stores temperatures in 5°C increments. Values are rounded to the nearest multiple of 5 during encoding. For example, 213°C will be stored as 215°C. This is by design and matches the spec.

How to edit a tag

First, read the tag using the Read tab. Then tap "Edit & Rewrite" next to the Scan button. This loads all the tag's data into the Write form where you can make changes. Tap "Write to Tag" and hold your phone near the tag to save.

Tag placement on spools

The OpenTag3D specification recommends two tags per spool, one on each end directly opposite each other, centered 56mm from the spool center. This allows automated readers mounted on either side of a printer to detect the spool.
